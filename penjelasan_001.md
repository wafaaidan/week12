Soal:
1. Definisi Variable
Jawab: Variable merupakan tempat untuk menyimpan angka.

2. Definisi class
Jawab: Class merupakan blueprint dari sebuah program untuk menyimpan beberapa method, yang mendefinisikan atribut (data/variabel) dan metode (perilaku/fungsi) yang akan dimiliki oleh objek-objek tersebut.

3. Penjelasan object dan penggunaan
Jawab: Object merupakan bentuk nyata dari blueprint (Class) yang memuat atribut-atribut dari class tersebut.

4. penjelasan method dan penggunaan
Jawab: Method adalah suatu kumpulan/ blok kode yang tidak mengembalikan nilai dan hanya menjalankan program yang ada di dalamnya.

5. penjelasan function dan penggunaan
Jawab: Function merupakan suatu blok kode yang tidak hanya menjalankan program melainkan juga mengembalikan nilai dan menyimpannya di dalam function itu sendiri.

6. penjelasan parameter dan penggunaan
Jawab: Parameter mengacu pada daftar variabel dalam deklarasi method. 
        Kemudian Argumen adalah nilai aktual yang diteruskan saat metode dipanggil sesuai urutan parameter yang dideklarasikan.
